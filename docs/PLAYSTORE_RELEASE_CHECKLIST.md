## 🔒 5. Deklarasi Data Safety (Keamanan Data)

Deklarasikan bahwa aplikasi Anda mengumpulkan data berikut dengan tujuan **Fungsionalitas Aplikasi** dan mengenkripsi data dalam transit:
1. **Nama & Email**: Untuk pendaftaran akun dan identitas profil.
2. **Nomor Telepon**: Untuk detail pengiriman belanja dan pesanan kurir.
3. **Alamat Fisik**: Untuk perhitungan ongkir otomatis dan tujuan kurir.
4. **Foto / Media**: Untuk mengunggah bukti pembayaran manual transfer, foto profil, dan lampiran posting feed.
5. **Data Chat / Diskusi**: Untuk komunikasi langsung antara buyer-seller.
6. **Riwayat Transaksi**: Untuk pelacakan invoice dan status pembayaran.

## 👮 6. Moderasi Konten User (UGC Moderation)

Google Play menerapkan aturan ketat terkait konten buatan pengguna (UGC):
* **Fitur Laporkan Konten (Report)**: Sudah tersedia di pojok kanan atas setiap postingan sosial media.
* **Fitur Sembunyikan Konten (Hide)**: Pengguna dapat menyembunyikan postingan yang tidak mereka sukai.
* **Moderasi Admin**: Admin/operator memiliki dashboard peninjauan aduan untuk menghapus postingan melanggar dalam waktu 24 jam.
